# 日文排版需求

本文简单整理了日语书写系统于排版上的需求。

## 日文排版的主要特色

日文的书写方向有直排及横排两种。

原则上，日文排版所使用的汉字、平假名和片假名比例皆为1:1的正方形，无缝隙并列排成版面。

本文参考了JIS X 4051标准和W3C《日文排版需求》文档。

本文的主要讨论对象是图书，但许多需求也适用于其他出版物。

## 参考文献
